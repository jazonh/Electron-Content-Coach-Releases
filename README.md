## üì± What is Electron Content Coach?

Electron Content Coach helps TCGplayer employees write communications that align with our brand voice and guidelines. Built with Electron, Next.js, and React Spring, it combines a modern UI with powerful AI capabilities.

### ‚ú® Features

- **AI-Powered Content Correction** - Uses GPT-5 Mini reasoning model for intelligent tone correction
- **RAG-Powered Guidelines** - Retrieves relevant brand guidelines using PyChomsky embeddings
- **Supabase Vector Search** - Fast, semantic search through TCGplayer brand guidelines
- **Target Audience Selection** - Choose between Buyer and Seller audiences
- **Multiple Variants** - Get 3 alternative versions with different approaches
- **Visual Processing Steps** - See the AI workflow in real-time with React Spring animations
- **Secure Credential Storage** - Environment-based configuration
- **Automatic Updates** - Seamless update system keeps you current
